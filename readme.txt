== Description ==

[Insert Adsense Code In Post](http://deltasoft.tn/wordpress/en/plugins/insert-adsense-code-in-post-wordpress-plugin) is a sample Wordpress plugin for adding the Google Adsense blocs in your Wordpress Posts or Pages.

== Installation ==

1. Extract `insert-adsense-code-in-post.zip` 
2. Upload `insert-adsense-code-in-post` directory to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently asked questions ==

= Therefore, it is advisable to use adsense blocks in 3 different postion right, top and bottom? =

Google Adsense statistics show that put your ads codes into your posts in 3 different postions can optimize and maximize your profit.
